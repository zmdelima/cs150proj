# CMSC 150 ULTIMATE OPTIMIZER
## Simplex Solver
### Setting the Objective
You can set the objective of the simplex by clicking on the MAXIMIZATION/MINIMIZATION button
### Objective Function
Set the objective function with the format: (value) * x1 + (value) * x2 + 0 
with 0 as the right hand side of the solution
### Adding/Removing Constraints
Create a new constraint at the bottom of the existing ones. 

Pressing the Remove Last Constraint 
takes out the last constraint that you have created
### Setting constraint Inequality
Set the specific constraint's inequality by clicking the "<=" or the ">=" 
button beside the constraint input box
### Solving and Outputs
The solver takes notice if you have blank constraints. 

Please fill up every constraint you created. After all is set press the Solve Button.

The solver will notify you on the bottom left corner if your objective function

and your constraints are infeasible
## Diet Optimizer
### Food Pages
There are 4 Pages containing each of it 16 foods in total to select and choose from for your diet
### Item Food Status Display
There is a table above which shows the current food clicked to display their status and variables
### Adding/Removing Foods
Add or Remove selected foods by cliking on the "+" to Add and "-" to Remove the food item from the plan

There is also a Clear button feature to clear out each choice in every page

and Choose All to select every food item for the diet plan
### Solution and Outputs
Click solve after choosing the preferred foods. The system will output a table containing the name,

serving, serving size, and its cost
